# 📊 Business Metrics Dashboard — FY 2023–24

A comprehensive business intelligence dashboard covering all core KPIs across marketing, sales, operations, HR, finance, and partner performance for the full financial year Oct 2023 – Sep 2024.

---

## 📸 What It Does

A single-page dashboard with interactive charts, month-on-month KPI tables, and an AI-powered insights panel — giving leadership and cross-functional teams a complete picture of business health at a glance.

---

## ✨ Features

- 📈 **35+ KPIs tracked** across 8 business categories
- 🗂️ **Category filters** — Marketing, Sales, Partner, HR, Finance, Operations, SF, Revenue
- 📅 **Month range filter** — Oct '23 through Sep '24
- 📊 **Interactive charts** — trend lines and bar charts per metric (Chart.js)
- 🧠 **AI Insights panel** — smart observations and recommendations powered by Claude
- 💬 **AI Chat** — ask natural language questions about the data
- 📋 **Full metrics table** — all KPIs month-on-month in one scrollable view
- 🌑 **Dark mode design** — IBM Plex Mono + IBM Plex Sans typography

---

## 📂 KPI Categories

| Category | Example Metrics |
|---|---|
| **Marketing** | CPL, CPA, Sessions, Leads, DTP Lead %, Metro Leads %, Conversion Rates CR1/CR2/CR3 |
| **Sales** | Total Sales, Total Devices, Gross Revenue, Net Revenue, Discount %, Sales Returns |
| **Partner** | Partner Accounts, Partner Centres, Active Show Partners, High Performing Partners |
| **HR** | Headcount, Employee Additions, Exits, Attrition Rate |
| **Finance** | Collections, Outstanding Debt, DSO |
| **Operations** | High-End %, BTE+RIC % |
| **Sales Force** | Sales Consultants count, Sales per SC |
| **Overall** | Revenue Target, Revenue Achievement % |

## ⚙️ Setup

1. Open `index.html` in any browser — all data is embedded in the file.
2. For AI Insights and Chat, ensure your Anthropic API key is configured in the script section.
